# Portfolio Design Plan

## 1. Portfolio Purpose
This portfolio is designed to showcase my UX/UI design skills for 2026, presenting in-depth, visual, and functional projects to potential employers.

## 2. Target Audience
- Startups seeking UX/UI design expertise
- Tech companies and design agencies
- Recruiters and HR professionals

## 3. Page Structure
- Home (Introductory overview)
- About (Experience, education, skills)
- Projects (In-depth case studies)
- Blog (Design trends and insights)
- Contact (Contact form, social links)

## 4. Color Palette
- Primary color: #0A2540 (Professional deep navy)
- Secondary color: #F0F0F0 (Light gray background)
- Accent color: #FF6F61 (Highlight)
- Text color: #1A1A1A (Dark gray)
  
## 5. Fonts
- Headings: Montserrat, Bold, 36px
- Body: Open Sans, Regular, 18px

## 6. Layout
- 12-column grid system
- Simple and clean
- Equal spacing between sections

## 7. User Flow
User lands on the home page, reviews the introduction, navigates to the about section, explores project case studies, and finally contacts via the contact page.

## 8. Responsive Design
- Mobile, tablet, desktop optimized
- Scalable navigation, images, and text

## 9. Tools Used
- Figma

## 10. Future Plans
- Add more case studies
- Develop interactive prototypes
- Expand blog section
