# Synthetic vs Real User Testing Worksheet

Persona: Səbinə, 52 yaş, müəllimə, beginner (A0–A1)  
Tapşırıq: İlk İngilis dili dərsini başlamaq və tamamlamaq  

---

## Müqayisə cədvəli

| Kriteriya                              | Synthetic                                                                 | Real                                                                                  | Fərq                                                                                     |
|----------------------------------------|---------------------------------------------------------------------------|---------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------|
| İlk reaksiya sürəti                    | Oxuyur, düşünür, sonra qərar verir (məs: 10 dəqiqə seçmək istəyir)       | Tez qərar verir, amma mənanı səhv başa düşür (20 dəq seçir, “azdır” deyir)           | Synthetic düşünərək qərar verir, real isə tez amma səhv anlayaraq qərar verir           |
| Frustrasiya nöqtələri                  | Dil sözlərini anlamır, amma davam edir                                    | Dilə görə çaşır, kömək istəyir, qərarı random verir                                   | Synthetic problemi keçir, real isə ilişir                                                |
| Emosional ton                          | Tərəddüd edir (mikrofon, səhv etmək)                                     | Həyəcanlanır, çəkinir, səhv edəndə utanır                                             | Real emosional reaksiya daha güclüdür                                                    |
| Gözlənilməz şərh / suallar             | Demək olar yoxdur, hər şeyi özü anlamağa çalışır                          | “Bu nədir?”, “Continue basım?” kimi suallar verir                                     | Real user daha çox sual verir                                                            |
| Mədəni / sosial faktorlar              | Müəllimə olduğu üçün səhvdən narahat olur                                | Səhv edəndə utanır və tərəddüd edir                                                   | Real-da bu faktor davranışa daha çox təsir edir                                          |
| Fiziki / davranış reaksiyaları         | Mikrofonla bağlı tərəddüd edir                                           | Danışanda çəkinir və həyəcanlandığını deyir                                           | Real-da fiziki/emotional reaksiya daha açıqdır                                           |
| App-dan kənar fikirlər                 | UI ilə bağlı düşünür (“bu nə deməkdir?” kimi daxili analiz)              | “Bu çətindir”, “utanıram” kimi şəxsi reaksiyalar verir                                | Real user daha şəxsi və emosional fikirlər deyir                                         |
| Səhv başa düşmələr                     | Bəzi sözləri anlamır, amma ümumi məntiqi tutur                           | Konsepti səhv anlayır (daily goal-u yanlış başa düşür, seçimləri anlamır)             | Synthetic qismən başa düşür, real isə ciddi səhv anlayır                                 |
| Feedback istifadəsi                    | “Explica mi error” düyməsinə baxır və istifadə etməyi düşünür            | İzahı oxumur, sadəcə “Continue” basır                                                 | Synthetic öyrənməyə çalışır, real isə öyrənmədən keçir                                   |
| Nəticə ekranı ilə davranış             | Çaşır, amma baxıb anlamağa çalışır                                      | Heç baxmadan “X” basır                                                                | Synthetic engagement göstərir, real isə tam ignore edir                                  |
